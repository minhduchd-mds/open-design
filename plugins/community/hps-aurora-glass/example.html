How · the Borealis loop

Forecasts that age in minutes, not days

Every site runs a 40-gram edge probe. Telemetry, sky-camera frames, and mesoscale weather fuse into one rolling forecast.

  • 15-minute re-training cadence per site
  • Sky-camera nowcasting for cloud edges
  • Dispatch API speaks operator protocols natively
  • Degrades gracefully — offline sites self-report
