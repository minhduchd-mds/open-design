Tokens

Same HTML, different soul

Every theme defines exactly the same variables. Switching themes never edits a slide — it swaps the sheet.

  • Colors: bg, surface, text-1/2/3, accent ×3
  • Semantics: good / warn / bad, grad / grad-soft
  • Shape: radius ×3, shadow ×2
  • Type: font-sans / display / serif / mono
:root {
  --bg        --surface   --surface-2
  --text-1    --text-2    --text-3
  --accent    --accent-2  --accent-3
  --good      --warn      --bad
  --grad      --radius    --shadow
  --font-sans --font-display
}

Comparison

Hand-rolled vs studio

Before

  • Charts are CSS, zero dependencies
  • Every deck speaks the same classes
